## 30/60/90 Day Optimization Rules

### Week 1–2: Learning Phase
**Touch nothing.** Let Google gather data.

### Week 3–4
- Review search terms report
- Add new negatives from irrelevant queries
- Pause keywords with 0 clicks and 50+ impressions

---

### Day 30 Review

| Condition | Action |
|---|---|
| Marin CPA < $200 | Increase Marin bid adjustment to +50% |
| Napa CPA > $400 | Pause Napa, reallocate budget to Marin |
| Sonoma CPA > $350 | Reduce Sonoma bid adjustment to -50% |
| Any keyword: 3+ clicks, 0 conversions | Pause the keyword |

### Day 60 Review

| Condition | Action |
|---|---|
| Overall CPA < $175 | Switch bidding to Maximize Conversions |
| Marin producing consistent leads | Increase daily budget to $25/day |
| Model proven | Begin planning scale to $1,500/mo |

### Day 90 Goals

- [ ] 1 signed contract from paid search
- [ ] CPA < $175 per lead
- [ ] 3–5 discovery calls booked total
- [ ] Clear winner keywords identified for scaling
